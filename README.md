# 🛡️ السلامة والصحة المهنية — HSE Immersive Learning World

> عالم تعليمي تفاعلي غامر ثلاثي الأبعاد في مجال السلامة والصحة المهنية
> إعداد: نورالدين الحكمي

## 📋 نظرة عامة

مشروع تعليمي متكامل يحتوي على **10 فصول شاملة** في مجال السلامة والصحة المهنية (HSE)، مبني بتقنيات الويب الحديثة مع تجربة مستخدم غامرة ثلاثية الأبعاد.

## 🏗️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| Next.js 15 (App Router) | إطار العمل الأساسي |
| React 19 | واجهة المستخدم |
| Three.js + React Three Fiber | مجسمات ثلاثية الأبعاد تفاعلية |
| Framer Motion | حركات وانتقالات سلسة |
| Zustand | إدارة الحالة (تتبع التقدم) |
| Tailwind CSS 4 | التصميم والأنماط |

## 📚 الفصول

1. التشريعات، قوانين العمل، والمخالفات الإدارية والقانونية
2. المخاطر الطبية والصحية، بيئة العمل، والتسمم الكيميائي
3. هندسة مسالك الهروب وأنظمة مكافحة الحرائق والعمل الساخن
4. هندسة السقالات، السلالم، وأنظمة الحماية من السقوط
5. هندسة الأوناش، الرافعات الشوكية، ووايرات الرفع
6. الأماكن المغلقة، أنظمة عزل الطاقة (LOTO)، وتصاريح العمل
7. السلامة الكهربائية، الأدوات اليدوية، والعدد الميكانيكية
8. تصنيف المناطق الخطرة، تحليل مخاطر الوظائف (JHA)، والتحقيق في الحوادث
9. مهمات الوقاية الشخصية (PPE)، بيئة العمل الصناعية، واللوحات التحذيرية
10. إصابات العمل، إدارة السلامة، والتأمينات الاجتماعية

## ✨ المميزات

- 🎮 **خريطة تفاعلية** لاستعراض الفصول مع مجسمات ثلاثية الأبعاد
- 📖 **شرح مفصل** لكل فصل مع تمييز المصطلحات الإنجليزية القابلة للنقر
- ❓ **بنك أسئلة وأجوبة** كامل لكل فصل (أكورديون تفاعلي)
- 🔢 **جدول الأرقام والضوابط** لكل فصل (الحدود القانونية والفنية)
- 📓 **دفتر مصطلحات** تفاعلي يحفظ المصطلحات التقنية
- 📊 **شريط تقدم** يحفظ إنجازك تلقائياً في المتصفح
- 🌙 **تصميم غامر** بخلفية فضائية داكنة وإضاءة نيون

## 🚀 التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على `http://localhost:3000`

```
hse-learning-world/

│   ├── globals.css       # الأنماط العامة + ثيم نيون
```
